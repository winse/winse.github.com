刷刷刷，装起 OpenCode，用上 MINIMAX

近一个月 Claude Code 几乎是如雷贯耳，Skill 的传播就像当年的 Prompts 提示词一样，旌旗招展，狂揽人气。

怀着对新事物的好奇，再加上国内编程大模型的突飞猛进，我也决定开个号，看多手痒，决定亲自试一试。

最近，最火要数 智普 和 MINIMAX 了，它们都在港股上市了。编程大模型就从它们两个中选了。

第一轮筛选，看公司股价，价高者得。

第二轮，看订阅费用，价优者得。

综合网上大家的评价来看：智谱的写代码能力强一点点，但是 MINIMAX Pro 的价格便宜了一半哦。

低头看了看账户余额，经过一番极致审慎的抉择😁，那还就选择 MINIMAX Starter 吧。

Open Code 直接用 npm 安装，一条命令搞定。

然后，按照 MINIMAX 的官方文档配置。

https://platform.minimaxi.com/docs/coding-plan/opencode

选一个目录，然后输入 opencode，稍等一会初始化完。

执行第一 opencode 的命令 /init 。https://opencode.ai/docs#initialize

整个过程很顺，交互界面虽说是命令行，但是体验和 IDE Chat 差不多。除了没有滚动条。

会话都是有保存的。按了 Ctrl + C 关掉后，下次再进行，可以选择继续之前的会话 Session。

接下来，给它一个体力活：重构 joda_time。它竟然给我搞出个 25-40 小时的计划。挺吓人，又挺感动，还挺汗人（这都是燃烧的 Token 啊）。

按 Tab 切换到 Build 模式，跑起来。

还是蛮经用的。

让它跑着，不用盯它。

先去学学 Skill，眯一会。

https://www.youtube.com/watch?v=yDc0\_8emz7M
