WIN 挂载 S3：像本地文件夹一样用对象存储

对象存储好用，但交互一直很「反人类」：上传还好，服务器处理的；但查看、下载必须进控制台或 Web 管理界面，找文件、翻目录、对比内容都非常不直观。

用 winfsp + rclone，可以直接把 S3 挂载成 Windows 本地磁盘，在资源管理器里像普通文件夹一样操作，体验瞬间升级。

# 环境准备

  • winfsp：https://winfsp.dev/rel/ 下载，下一步直至安装完成。
  • rclone：https://rclone.org/downloads/ 下载，解压。

# 一、配置 S3

rclone config
  1. 输入 n (New remote)。
  2. name：输入 rustfs_remote（可以自定义）。
  3. Storage：选择 s3，输入数字 4 (视版本而定)。
  4. provider：选择 Other，输入数字 47。
  5. env_auth：选择输入 false。
  6. access_key_id：输入你在 RustFS 设置的 Access Key：minioadmin。
  7. secret_access_key：输入对应的 Secret Key：minioadmin。
  8. region：默认的为空即可，直接回车。
  9. endpoint：输入你的 RustFS 服务器地址，输入 http://127.0.0.1:9000。
  10. location_constraint：默认的为空即可，直接回车。
  11. acl：一般选 private 或留空，输入 private。
  12. 其他高级选项直接按 Enter 跳过。
  13. 最后确认 y，或者直接回车。

# 二、挂载

rclone mount rustfs_remote:test2 Z: --vfs-cache-mode full --links

挂载完成后，Z: 盘就是你的 S3 本地磁盘。

# 三、使用体验

  • 可直接拖拽上传
  • 支持复制、删除、重命名
  • 可用任意软件直接读写

# 体验总结

S3 + 文件系统 = 生产力翻倍。对象存储终于变成易用、好用、近乎透明的形态了。
