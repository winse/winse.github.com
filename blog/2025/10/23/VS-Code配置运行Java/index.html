VS Code配置运行Java

好久不写 Java 代码了，用 VS Code 生成 Java 代码还要再切到 Eclipse 去运行，感觉非常之别捏。

原来也看过在 VS Code 配置 Java 环境，但终究觉得插件麻烦而作罢。现在是 AI 给我生成了 Java 代码，要切到 Eclipse 运行觉得麻烦了。人啦，就是这样，反反复复，麻麻烦烦。

配置跑起来后，其实 VS Code 跑 Java 代码体验也很丝滑，配置也很简单。

  1. 安装 Java 插件：Extension Pack for Java，其他依赖插件也会一次性全部装好；
  2. 配置 Gradle：把 Wrapper 禁掉，同时指定 GRADLE_HOME 和 JAVA_HOME；
  3. 当在 JAVA PROJECTS 视图中看到 Java 工程了，就说明 Java 配置成功了。

后面就可以直接在编辑器中运行、调试 Java 代码了。

一代新人换旧人啊，VS Code 能文能武，真好。
