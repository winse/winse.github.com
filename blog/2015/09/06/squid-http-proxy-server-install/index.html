安装http代理服务器squid

# 环境说明

  • squid-3.3.14.tar.gz
  • centos6.6

# 安装

yum install gcc gcc-c++
cd squid-3.3.14
./configure
make
make install

cd /usr/local/squid
#不修改会有权限的问题
chmod 777 var/logs
sbin/squid 
sbin/squid -k shutdown

或者：

yum -y install squid
chkconfig squid on

改下squid.conf的配置：

# And finally deny all other access to this proxy
#http_access deny all
http_access allow all

# 使用

在浏览器中设置Http代理。端口为3128
