安装

  • Termux is an Android terminal app and Linux environment.
  • Android下的Linux环境和Shell终端程序，无须root，可使用apt包管理工具安装大部分Linux软件。
  • 国内下载通过官网的 F-Droid ，该网站安装包的各个版本描述下一个 下载APK 的链接。版本 0.72
  • Android源码：termux-app
